# 🚀 Ibov - Node.js

## Available Scripts

Runs the app in the development mode.\
Open [http://localhost:3000](http://localhost:3000) to view it in the browser.

## 📁 How to download the project

```bash
$ git clone https://github.com/LanPRD/ibov-backend-nodejs.git

$ cd IBOV

$ npm install

$ npm start
```
